ENDLESS SUMMER "GAUNTLET" (DUE NEXT WEEK)

Create a small game based on some aspect of summertime. The thematic details, goals, and mechanics are up to you. Keep it brief & manage your time!

Technical requirements:

  • Built using Unity
  • Multiple scenes
  • Canvas-based UI
  • Input through Input Manager (not GetKey*)
  • Simple animation. At least one instance controlled through code
  • Rigidbody2D collision/trigger
  • Use of sound
  • Use of IEnumerator coroutine
  • Exported build for your OS of choice

We will play these games at the beginning of class next week. Please create a tested build which can run on your computer. Bring a controller if necessary.
