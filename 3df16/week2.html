UNITY 3D: INTERFACE

Go over some important parts of the editor, relevant to 3D:

  • Scene view toggles: Shading mode, 2D/3D, lighting, skybox
  • Scene navigation: Local/Global rotation, 3D view gizmo, Focus, Align to View (Camera)
  • Lighting panel: Add it via Window menu

UNITY 3D: PRIMITIVES

Unity has a small selection of primitive forms which can be created in-editor*:

  • Planes, cubes, spheres, cylinders, capsules
  • Adding, manipulating, focusing in scene
  • *More can be found on the Asset Store. (Primitive Plus is $5)

UNITY 3D: LIGHTING

A look at some of Unity's basic realtime lighting options:

  • GI: Global Illumination
  • Basic types: Directional, point, spotlight
  • Positioning, color, intensity, casting shadows

UNITY 3D: COLOR MATERIALS

Adding color to an object using a custom material:

  • Create in Project: New Material
  • Change color(Albedo), reflection, and smoothness
  • Choose new material in an object's Mesh Renderer component (Material 0)
  • Further editing can be done from object, but it will affect all others using that material

UNITY 3D: SKYBOX

The skybox can be manipulated, or removed, in the Lighting panel:

  • Open Lighting > Scene
  • Under Environment Lighting you have the ability to change the Skybox and ambient lighting
  • New Skyboxes are added as textures, or procedurally created, as if a Material (Choose Skybox as type instead of Standard)

CHIAROSCURO (DUE START OF CLASS NEXT WEEK)

Chiaroscuro is the use of contrasts of light to achieve a sense of volume in modelling three-dimensional objects and figures.

Working with only un-textured primitives and lighting, create a virtual form/structure/sculpture that can be explored by moving around, under, or through it.

Technical requirements:

  • Greyboxing: Work fast, in-engine assets only (Primitive Plus Unity Asset or equivalent is OK too, but no imported models!)
  • Think structurally, not spatially, for this one.
  • Use of colored materials is OK
  • Work from sketches
Some possible inspiration:
  • Chiaroscuro techniques in fine art & film
  • Large-scale modernist sculpture such as Tony Smith's Moondog (1964) or Richard Serra's Torqued Ellipse IV
  • Architectural use of light & shadow such as these
  • Games such as Echochrome 2, Kairo, Unfinished Swan, etc.
